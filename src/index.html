Projetos

Mallow

Mallow foi o jogo que desenvolvi durante o curso PlayGame.
Fui o responsável por todo o processo de desenvolvimento desde o planejamento visual, mecânicas e narrativa até a implementação e programação na Unreal Engine.
Este foi meu primeiro contato direto com lógica de programação, usando Blueprints, o sistema visual da engine.
Por isso, além de criar a estética e a história do jogo, um dos maiores desafios foi justamente desenvolver as mecânicas e fazer tudo funcionar na prática.

SAMA

O SAMA – Sistema de Atestados Médicos e Avaliações foi o projeto que minha equipe, .Tec, desenvolveu no primeiro semestre da faculdade de ADS.
Usamos o framework Flask no backend e, no frontend, trabalhamos com JavaScript, HTML, CSS e Bootstrap. Organizamos o time com base no método Scrum, e eu atuei como desenvolvedor.
Minhas principais responsabilidades foram:
> Criar um dashboard com métricas de aprovação e reprovação de atestados, voltado para o controle da secretaria;
> Implementar a identificação automática do motivo do afastamento, com base no código CID do atestado, usando a Google Charts para visualização dos dados;
> Desenvolver a página de perfil do usuário, com funcionalidades para editar dados pessoais, excluir a conta e visualizar as equipes das quais ele faz parte;
Trabalhar no backend da área de Equipes Ágeis, responsável pelo envio das avaliações feitas com o método Pacer e pelo dashboard de desempenho da equipe.
Foi meu primeiro projeto em equipe, e, por isso, rolaram alguns desafios na comunicação e organização. Mas conseguimos superar tudo e entregar um sistema bonito, funcional e robusto, mesmo usando apenas arquivos .txt como banco de dados.

RunRat

RunRat é um jogo que criei no Scratch onde o jogador controla um rato tentando escapar de vários gatos em um espaço limitado.
Desenvolvi o jogo como um passatempo, com o objetivo de treinar minha lógica de programação com blocos.
O maior desafio foi ajustar as mecânicas de movimentação e colisão para que o rato e os gatos se não atravessassem as paredes, para manter o jogo desafiador.
